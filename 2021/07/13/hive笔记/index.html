一、数据类型和文件格式

1、基本数据类型

Hive支持多种不同长度的整型和浮点型，支持布尔类型，也支持无长度限制的字符串类型。Hive 0.8.0中增加了时间戳数据类型和二进制数组数据类型。

数据类型 长度 例子
TINYINT 1byte 有符号整数 20
SMALINT 2byte 有符号整数 20
INT 4byte 有符号整数 20
BIGINT 8byte 有符号整数 20
BOOLEAN true 或 false TRUE
FLOAT 单精度浮点数 3.14159
DOUBLE 双精度浮点数 3.14159
STRING 字符序列 ‘now is the time’
TIMESTAMP (v0.8.0+) 整数，浮点数或者字符串 1327882394(Unix新纪元秒)
BINARY (v0.8.0+) 字节数组
2、集合数据类型

Hive中的列支持使用struct、map和array集合数据类型。

数据类型 描述
STRUCT 可以通过“点”符号访问内容，例如某个列的数据类型是STRUCT{first STRING, last STRING}，那么第1个元素可以通过字段名.first来引用
MAP 是一组键值对元组集合，使用数组表示法可以访问元素
AARRAY 是一组具有相同类型和名称的变量的集合，每个数组元素都有一个编号，从0开始
